# 八二共识


## 推广策略

-	布道价值、促进共识
-	面对任何一个新区域、新群体、新市场
-	必须有两个角色成对出现：社区领袖和布道者（传道士、牧师）
-	布道时间 : 销售时间 = 8 : 2
-	社区领袖 + 布道者，捉对合作，成对出现
-	布道者讲清VOKEN三件事，讲清楚商业逻辑和价值本质，最好能再讲讲技术
-	销售型领袖画龙点睛
